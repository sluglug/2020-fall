Workshops

---------
- A computer with a Unix environment, e.g. macOS, Linux

Not required:
-------------
- Prior knowledge or experience

Topics
======
- Command line basics
- Changing your environment
- Shell scripting
- Using pipelines
- Regular expressions
- vi and vim
- Customizing and extending vim
- Documents with troff/pandoc
- Getting help and learning more
